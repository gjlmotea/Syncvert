# Syncvert: Curl → yt-dlp 即時轉換工具

將 `curl` 指令即時轉換為 `yt-dlp` 指令，並透過 Socket.IO 實現多人同步協作。

---

## 🚀 如何運行

### 後端（Node.js + Socket.IO）

```bash
cd server
npm install
npm run server
```
預設啟動於：http://localhost:3001

### 前端（React）

```bash
cd client
npm install
npm run dev
```

預設啟動於：http://localhost:5173
